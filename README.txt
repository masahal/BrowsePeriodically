Every hour, day, week and so on, this add-on browses sites you want to check periodically such as web mail, forecast, RSS reader, SNS... constantly being updated.
You can also use for reminding pages by browsing every month.

You can set browsing intervals by the hour.

定期的にチェックしたいページ――例えば Web メール、天気予報、RSS reader、SNSなどを、1時間ごと、1日ごと、1週間ごとというように定期的に表示します。

役に立つ情報が載ってるけどすぐには使わないページをリマインダー代わりに1ヶ月ごとに表示する、といった使い方もできます。

表示間隔は1時間単位で設定できます。
